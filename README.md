# Amelia Liw's Portfolio Website

Portfolio showcasing UX/Product Design work, built with Next.js.

## Tech Stack
- Next.js 15 with App Router
- TypeScript  
- Tailwind CSS
- Deployed on Vercel

## Featured Case Studies
- **StoreHub Membership** (2024-2025): Cross-platform loyalty system
- **AI Design Review Clone** (2024): Process automation tool
- **BackOffice Design System** (2022): Component library and guidelines
- **Earlier Projects**: KTM Transit and SupaBolt Data Platform

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
